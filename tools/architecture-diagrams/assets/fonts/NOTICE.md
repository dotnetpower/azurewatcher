# Diagram Font

`noto-sans-kr-diagrams.ttf` is a subset of Noto Sans KR from the
[Google Fonts repository](https://github.com/google/fonts/tree/main/ofl/notosanskr).
Noto Sans KR is licensed under the SIL Open Font License 1.1.

The subset contains the Latin range and glyphs used by the bilingual diagram
specifications. `font.lock.json` records the source and subset checksums. Update
the subset whenever a diagram adds a character that isn't present in the
current file.
