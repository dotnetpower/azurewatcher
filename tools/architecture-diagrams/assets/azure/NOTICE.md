# Azure Architecture Icons

This directory contains an allowlisted subset of the official Azure Architecture
Icons V24 package. The diagram compiler embeds these files without cropping,
rotation, recoloring, or shape changes.

Microsoft permits these icons to be copied, distributed, and displayed in
architecture diagrams, training materials, and documentation. Other use is not
included. Review the current terms on the
[Azure Architecture Icons](https://learn.microsoft.com/azure/architecture/icons/)
page before updating this subset.

`icons.lock.json` records the source archive and SHA-256 digest for every
included file. Builds run without downloading icon assets from the network.
